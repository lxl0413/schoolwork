【可视化排序程序运行指南】

【数据输入】
在files文件夹中的input.in文件中输入待排序数据。
格式为<数据类型> <待排序数据>
数据类型有int、double、float三种。
例如：double 2.1 4.2 5 6.7 8.6 
我们提供了三个inputexample文件，作为输入样例。

【程序运行】
点击VisibleSort.exe运行程序。
在开始界面选择排序类型后，点击Start按钮进入排序界面。
进入排序界面后，点击Play按钮开始播放排序动画，再次点击可以暂停。
暂停状态下，可以进行单步执行操作，也可以按Save保存快照。

【读取快照】
在开始界面点击Load，可以读取保存的快照并继续运行。

【退出程序】
点击Quit按钮退出。
